Bienvenido al Juego Toque y Fama Version JavaScript




El juego consiste en lo siguiente...

  • Debes seleccionar el largo del numero (Este el maximo numero de intentos que tendras para acertar)
  • El juego consta de que tienes que adivinar un codigo secreto con un x largo de numero
  • Una fama ocurre cuando le acertas a un numero que esta en el codigo secreto
  • Un toque ocurre cuando le acertas a la posicion de uno o mas numeros dentro del codigo secreto

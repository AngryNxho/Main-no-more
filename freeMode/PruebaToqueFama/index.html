Bienvenido al Juego Toque y Fama Version JavaScript




El juego consiste en lo siguiente...

  • Debes seleccionar el largo del numero (este sera la cantidad de vidas que tengas)
  • El juego consta de adivinar un codigo secreto con un x largo de numeros
  • Un Toque ocurre cuando le aciertas a un numero que esta en el codigo secreto
  • Una Fama ocurre cuando le aciertas a la posicion de uno o mas numeros dentro del codigo secreto
